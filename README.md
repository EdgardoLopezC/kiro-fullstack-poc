# Kiro Fullstack POC

Proyecto de prueba técnica con stack completo.

## Stack Tecnológico

- **Frontend**: Angular (Standalone Components)
- **Backend**: Quarkus (Java 17)
- **Base de Datos**: MySQL

## Estructura del Proyecto

```
/frontend   - Aplicación Angular
/backend    - API Quarkus
```

## Configuración

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
./mvnw quarkus:dev
```

## Desarrollo

Este proyecto sigue Git Flow estándar con commits en la rama `main`.
